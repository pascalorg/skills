# Pascal Skills

A collection of agent skills by [Pascal](https://github.com/pascalorg).

## Install

```bash
# Install all skills
npx skills add pascalorg/skills

# Install a specific skill
npx skills add pascalorg/skills --skill image-analysis
```

## Available skills

| Skill | Description |
|-------|-------------|
| [image-analysis](./image-analysis) | Extract color palettes from images (screenshots, Figma exports, design mockups) to help implement matching UI |
